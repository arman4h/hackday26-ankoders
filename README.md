## üí° Our Solution

### **Silent Bridge ‚Äì A Non-Verbal Classroom Communication System**

Silent Bridge is a simple, web-based solution that enables **non-verbal communication between students and teachers**.  

The system allows students to express common classroom needs silently using predefined options, such as:

- Requesting to go to the restroom
- Indicating they are not feeling well
- Expressing confusion about a topic
- Confirming understanding of the lesson
- Requesting help from the teacher

By interacting with the interface, students can communicate without speaking, raising hands, or disrupting the class.
